# react-native-weather-app

1. `git clone`
2. `cd react-native-weather-app`
3. `node -v`
4. `npm -v`
5. `npm install -g expo-cli`
6. `npm install`
7. `npm start`
8. download expo client on your mobile device
9. plug device
10. enable developmer mode
11. use command to run on device (`a` for Android)
